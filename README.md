## 🧠 Core Architecture: 7-Kernel Orchestra
DROS는 단순한 트레이딩 봇이 아닙니다. 16개의 특화된 AI 에이전트가 실시간으로 데이터를 수집, 추론, 검증, 실행하는 **자율형 운영체제(Autonomous OS)**입니다.

* **ToxicityShield (Game Theory):** VPIN과 OFI 데이터를 분석하여 기관의 역선택(Adverse Selection)을 감지하고, 폭락 직전 진입을 차단합니다.
* **Vampire Capital Allocator:** 정체된 포지션의 자본을 회수하여 새로운 알파(Alpha) 창출에 실시간 재배분합니다.
* **AI Evolution Lab:** 시장이 변하면 시스템도 진화합니다. 자체적인 OODA 루프와 디지털 트윈(Digital Twin)을 통해 스스로 전략을 백테스트하고 승급시킵니다.
* **Zero-Overhead Execution:** Apple MLX 프레임워크와 SharedMemory 기반의 극단적인 로컬 하드웨어 최적화.
